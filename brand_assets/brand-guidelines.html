Brand Mark

Logo Usage

The LK Nail Salon logo must always appear with adequate clear space and on approved backgrounds only.

Primary — Dark Background Use on dark/black backgrounds. This is the preferred treatment.
On Gold Gradient Acceptable for special promotions and seasonal campaigns.
Light Background — Dark Version Use a dark/black version of the logo on light backgrounds.
✗
Never — Patterned or Busy Background Do not place the logo on complex textures or competing imagery.

Clear Space Rule

Always maintain a minimum clear space equal to the height of the crown symbol on all four sides of the logo. Never crowd the logo with other elements.

Usage Rules

Do & Don't

✓ Do
  • Use Cardo for all headings and display text
  • Pair Cardo headings with Lato body copy
  • Use the signature gold gradient for primary CTAs
  • Place the logo on dark (#1B1D1C) or gold gradient backgrounds
  • Apply tight letter-spacing (–0.03em) to large headings
  • Layer shadows with brand-tinted colors, not flat grey
  • Use kicker labels (uppercase Lato) before every section heading
  • Only animate transform and opacity
  • Provide hover, focus-visible, and active states on all interactive elements
✗ Don't
  • Use blue, purple, teal, green, or generic default palette colors
  • Use the same font for both headings and body text
  • Use transition-all — always specify exact properties
  • Place the logo on busy, patterned, or low-contrast backgrounds
  • Stretch, rotate, recolor, or crop the logo mark
  • Use flat single-color shadows (shadow-md without color tint)
  • Use inconsistent spacing — always reference the spacing tokens
  • Mix multiple script or display fonts — Dream Avenue is logo-only
